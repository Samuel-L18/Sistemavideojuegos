# Sistema de Videojuegos

Aplicación de consola en C# para gestionar los implementos del sistema: jugadores, videojuegos y partidas.

## Funcionalidades
- CRUD de Jugadores
- CRUD de Videojuegos
- CRUD de Partidas
- Persistencia en CSV usando CsvHelper

## UML
El diagrama se encuentra en el archivo `uml.mmd`.
https://drive.google.com/file/d/17L9XGA2ETWBN4N2K1R_p480s-HP9dcn3/view?usp=sharing
## Requisitos
- .NET 6 o superior

## Instalación
```bash
dotnet restore
dotnet run
```
